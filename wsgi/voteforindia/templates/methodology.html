How do we rank?

WhoGetsMyVote.in serves as an inspiration for us to create this app. We use the same parameters as they do, but we might differ in calculating the percentage for every parties to match with your ideology. Because, we do not know how they calculate, since we do not have access to their server side code. Here is the link to their methodology

We have made this app a Free Software under GPL v3 License, which means we wanted to be more transparent about how we calculate. The source code for this site is available online and anyone who wishes to see how we perform operations are most welcome. Do not stop there, if you find any flaw in our percentage calculation or if you feel we can do better, please feel free to fork the repo and modify the code or create issues or email us at "ask@fsftn.org"

We are constantly reading the manifesto of the 5 Parties (INC, CPI, CPM, BJP & AAP) and tuning our parameters accordingly. So expect changes in results as we progress with manifesto.

Apps like this should be trustworthy. To be trustworthy, we feel it should be transparent. To be transparent, the source code should be in public domain. This is the reason we created our own version of the same app.
